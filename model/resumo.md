CSS
---

Imagem (img):
- width (largura)
- height (altura)
- border-radius (arredondar os cantos)
- object-fit (garantir que a imagem preencha o espaço sem distorcer)

Título (h1):
- text-align (centralizar o texto)

Container (#container):
- display: flex (ativar modo flex)
- justify-content: center (centralizar horizontalmente)
- align-items: center (centralizar verticalmente)
- gap (espaçamento entre os elementos filhos)
- flex-wrap (quebrar linha se necessário)
- max-width (largura máxima do container)
- margin: 0 auto (centralizar horizontalmente)
- text-align (centralizar texto interno, se houver)
